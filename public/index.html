AGAPAY Learn

Orthodox homeschool planning for families.

AGAPAY Learn is an Orthodox homeschool planning app for families. Parents use AGAPAY Learn to organize lessons, terms, books, printables, reports, and liturgical calendar rhythms in one household planner.

Google Calendar Sync

If a family chooses to connect Google Calendar, AGAPAY Learn asks permission only to create or sync homeschool planning events to that family's calendar. Calendar sync is optional. AGAPAY Learn does not sell Google user data and does not use Google Calendar data for advertising.
